## JIT卓越工程师网站资源下载

### 使用方法

1.安装Python环境

2.确保网络为**校园内网**

3.双击运行**Main.py**

### 注意事项

因为密码输入使用了getpass模块，所以请**不要在IDE中运行程序**，双击运行就可以了。

**请不要将获取的资源对外传播。**
