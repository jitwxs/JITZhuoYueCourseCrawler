## 一、起步

（1）安装 `Python 3.X`，并安装以下第三方库：

```bash
pip install requests
pip install bs4
```

（2）将网络切换到 `JIT-EDU`，或其他校园内部网络，确保能够正常访问课程主页：`http://223.2.193.200`。

（3）双击运行 `Main.py`
